# 住建部通报建筑施工转包违法分包等违法行为典型案例（2014）

## 案情简介
住房城乡建设部在专项检查中查实以下施工单位存在转包、违法分包等市场违法行为：

- **转包（人员"假自营"）**：某工程施工单位存在转包行为，项目经理及主要管理人员的劳动合同与其下属子公司签订，社保由下属子公司缴纳，工资由下属子公司支付；建设单位未取得施工许可证开工建设，未按合同约定支付工程款；专业分包和劳务分包单位的现场负责人无劳动合同、社保凭证、工资支付凭证。
- **齐齐哈尔北疆经典小区一期工程（案例九）**：齐齐哈尔北疆建筑安装工程有限公司施工的北疆经典小区一期一标段2-4号楼工程，施工单位存在转包行为，未对项目施工实施有效管理，将主要建筑材料、构配件、工程设备的采购交由劳务分包单位现场负责人实施，土石方分包工程也由该负责人实施。
- **庆阳南郡壹號工程（案例十）**：甘肃第五建设集团公司施工的南郡壹號2号、3号住宅楼及地下车库工程，施工单位存在转包行为，不履行管理义务，只收取管理费，现场主要管理人员无社保证明。
- **海东平安东方明珠都市花园工程（案例十一）**：青海省广厦建筑工程有限公司施工的9号、10号、11号楼项目，一是施工单位存在转包行为，将其承包的10号、11号楼工程转给非本公司注册项目经理等人施工；二是施工单位存在违法分包行为，将防水分部工程分包给无资质的自然人个人；三是9号楼现场主要管理人员无社保证明。

## 处理结果
通报要求：存在工程质量问题和市场违法行为的单位要认真进行整改；工程所在地住房城乡建设主管部门要对整改情况进行跟踪检查；省级住房城乡建设主管部门要组织对上述施工单位在本省（区、市）承建的其他工程项目进行检查，并将检查情况、整改情况和处理意见逐级上报住房城乡建设部。

## 警示意义
本批案例集中揭示了施工单位"挂靠式转包""收取管理费不管理""将材料采购与分包交由个人"等典型转包、违法分包手法。案例明确：母公司承接工程后将所承接工程交由具有独立法人资格的子公司施工、施工单位只收管理费不实施管理、主要材料与分包由个人实施等，均属转包或违法分包。各施工单位应严格依法组织施工，切实履行合同管理义务，各级主管部门应依法认定并严肃查处。
