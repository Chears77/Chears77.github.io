# 财建〔2022〕183号《关于完善建设工程价款结算有关办法的通知》政策解读

## 背景概述

2022年6月30日，财政部、住房城乡建设部联合发布《关于完善建设工程价款结算有关办法的通知》（财建〔2022〕183号），自2022年8月1日起施行。该文件在《建设工程价款结算暂行办法》（财建〔2004〕369号）的框架下，针对工程价款结算中长期存在的进度款支付比例偏低、竣工结算久拖不决等突出问题提出两项重大改革措施：**提高进度款支付比例至不低于80%** 和**推行施工过程结算**。

此前，财建〔2004〕369号规定工程进度款支付比例为60%-90%，实践中很多政府项目压至60%甚至更低，导致施工企业大量垫资，农民工工资拖欠问题反复发生。建筑业产值利润率长期低下（约2%-3%），工程款拖欠是主要原因之一。

---

## 核心内容一：提高建设工程进度款支付比例

### 政策原文

> 政府机关、事业单位、国有企业建设工程进度款支付应不低于已完成工程价款的80%；同时，在确保不超出工程总概（预）算以及工程决（结）算工作顺利开展的前提下，除按合同约定保留不超过工程价款总额3%的质量保证金外，进度款支付比例可由发承包双方根据项目实际情况自行确定。在结算过程中，若发生进度款支付超出实际已完成工程价款的情况，承包单位应按规定在结算后30日内向发包单位返还多收到的工程进度款。

### 要点解析

**1. 适用范围**

"政府机关、事业单位、国有企业"作为发包人的建设工程。非国有投资项目（如民营开发商开发项目）不受80%下限约束，但可参照执行。

**2. 80%是最低比例而非固定比例**

文件明确"不低于80%"，实际支付比例可由发承包双方根据项目实际情况自行确定，可高于80%（如在确保不超出概预算的前提下按85%、90%支付），但不得低于80%。

**3. 3%质量保证金上限不变**

质量保证金预留比例仍为不高于工程价款总额的3%，延续了财建〔2004〕369号和《建设工程质量保证金管理办法》（建质〔2017〕138号）的规定。

**4. 超付返还机制**

若进度款支付超出实际已完成工程价款，承包单位应按规定在结算后30日内返还多收部分，建立了超额支付的纠偏机制。

### 新旧对比

| 事项 | 财建〔2004〕369号（旧） | 财建〔2022〕183号（新） |
|------|------------------------|------------------------|
| 进度款支付下限 | 60% | **80%** |
| 适用发包人 | 全部 | 政府机关/事业单位/国有企业 |
| 质量保证金上限 | 3%或5% | **3%** |
| 超付返还 | 无明确规定 | 结算后30日内返还 |

### 对工程实践的影响

- **施工企业现金流显著改善**：进度款从60%提升至80%，企业垫资压力减少20个百分点。以一个合同价1亿元的项目为例，每次进度款支付可多获得约200万元现金流。
- **农民工工资保障加固**：进度款的及时足额支付是农民工工资按时发放的资金来源基础。80%的支付比例意味着"钱跟人走"的资金链条更加稳定。
- **违规成本**：发包人不按不低于80%比例支付进度款的，可能面临行政监督和审计监督的双重压力。

## 核心内容二：推行施工过程结算

### 政策原文

> 当年开工、当年不能竣工的新开工项目可以推行过程结算。发承包双方通过合同约定，将施工过程按时间或进度节点划分施工周期，对周期内已完成且无争议的工程量（含变更、签证、索赔等）进行价款计算、确认和支付，支付金额不得超出已完工部分对应的批复概（预）算。经双方确认的过程结算文件作为竣工结算文件的组成部分，竣工后原则上不再重复审核。

### 要点解析

**1. 过程结算的适用条件**

- 对象：**当年开工、当年不能竣工的新开工项目**（跨年度工程）
- 方式：**可以推行**（非强制，但鼓励）
- 关键前提：**通过合同约定**（须在签约时明确约定过程结算的安排）

**2. 过程结算的操作机制**

```
施工过程 → 按时间/进度节点划分施工周期 →
周期内已完成且无争议的工程量（含变更、签证、索赔）→
价款计算、确认和支付 →
过程结算文件 = 竣工结算文件的组成部分 →
竣工后原则上不再重复审核
```

**3. 过程结算的范围**

包括周期内的：
- 已完成的合同内工程量
- 工程变更（设计变更、现场签证等）
- 索赔事项（工期、费用）
- 其他已确认的价款调整

"无争议"是关键限定——有争议的部分不纳入当期过程结算，另行处理。

**4. "竣工后原则上不再重复审核"的制度意义**

这是过程结算制度的核心价值所在。传统的"一次竣工结算"模式下：
- 施工过程中的变更、签证、索赔等全部积累到竣工后一次性处理
- 竣工结算往往延续半年甚至数年
- 结算争议引发大量诉讼

过程结算将结算工作前移至施工过程中，分阶段"清账"，竣工时只需汇总各期过程结算文件，大幅缩短竣工结算时间。

**5. 支付金额上限**

支付金额不得超出已完工部分对应的批复概（预）算，防止超概支付。

### 过程结算制度的价值

| 维度 | 传统竣工结算 | 过程结算 |
|------|------------|---------|
| 结算频率 | 竣工后一次性 | 按周期分阶段 |
| 争议处理 | 累积爆发 | 当期化解 |
| 竣工结算时间 | 数月甚至数年 | 原则上不再重复审核 |
| 企业资金周转 | 竣工前大量垫资 | 施工过程中持续回款 |
| 农民工工资保障 | 间接保障 | 更直接保障 |

### 各地配套推进

财建〔2022〕183号发布后，各省纷纷出台配套细则：

| 省份 | 配套文件 | 特别规定 |
|------|---------|---------|
| 湖北省 | 鄂建文〔2022〕27号 | 工程质量保证金降至**1.5%**，鼓励以银行保函替代 |
| 广东省 | — | 明确过程结算适用于工期1年以上的项目 |
| 浙江省 | — | 过程结算与BIM技术协同推进 |
| 多省 | — | 将过程结算纳入招标文件和施工合同示范文本 |

## 两制度协同效应分析

进度款提高至80%与过程结算的协同推进，形成了完整的"资金链-工程进度"良性循环机制：

```
过程结算（缩短结算周期）
    → 过程价款确认与支付（减少争议积累）
    → 进度款不低于80%支付（改善现金流）
    → 农民工工资及时发放（末端保障）
    → 减少因资金问题导致的停工和纠纷
    → 工程按计划推进
    → 下一周期过程结算...
```

## 实践注意事项

### 发包人（建设单位）

1. **合同约定前置**：过程结算须在施工合同中列明节点划分、结算程序、审核时限等条款，事后补充困难较大。
2. **概预算控制**：过程结算支付金额不得超出概预算，建议在合同中约定各节点的概预算控制限额。
3. **超付管理**：发现超付应及时要求承包人在30日内返还，避免资金闲置或挪用。

### 承包人（施工单位）

1. **及时申报**：按约定节点及时提交过程结算资料，避免因拖延申报而丧失当期收款权。
2. **争议处理**：对发包人不予确认的部分，应在当期明确记录争议事项，避免争议累积。
3. **农民工工资分账管理**：利用提高的进度款支付比例，确保农民工工资专用账户资金充足。
4. **竣工结算简化**：妥善保存各期过程结算文件，竣工时汇总即可，争取"不再重复审核"的制度红利。

### 监管部门

1. 对政府投资项目发包人是否按不低于80%支付进度款进行监督。
2. 鼓励在政府投资工程中率先推行过程结算。
3. 将进度款支付和执行过程结算情况纳入建筑市场信用评价。

| 法规文件 | 衔接关系 |
|---------|---------|
| 《保障农民工工资支付条例》（国务院令第724号） | 183号文提高进度款支付比例为条例的"工资专户"制度提供更充足的源头资金 |
| 《建设工程价款结算暂行办法》（财建〔2004〕369号） | 183号文在369号基础上上调进度款比例下限，其他事项继续按369号执行 |

| 建工解释一（法释〔2020〕25号）第27条 | "利息从应付工程价款之日计付"的规定与本通知共同构成完整的价款支付保障体系 |

## 实施效应评估

截至2024年的实践反馈：

**积极效应**：
- 政府投资工程进度款拖欠投诉量明显下降
- 多地反映"竣工后原则上不再重复审核"有效缩短结算时间
- 施工企业现金流状况改善，中小建筑企业尤为受益

**存在问题**：
- 非政府投资项目执行动力不足
- 部分地区过程结算节点划分标准不统一
- 变更、签证、索赔的"无争议"确认机制有待完善
