# 住建部深化建筑业改革与建筑市场监管最新政策解读——资质管理改革、招标投标整治与建筑法修订动态

## 政策背景

建筑业是我国国民经济支柱产业。面对新形势新任务，住房和城乡建设部持续推进建筑业改革，围绕法治建设、市场监管、营商环境三大主线，加快构建"宽进、严管、重处"的建筑市场监管新机制。

本文综合梳理住建部2025年建筑市场监管司政策导向、《关于进一步完善建筑市场监管机制持续优化建筑市场环境的通知（征求意见稿）》以及各省资质管理改革实践，对当前建筑业改革重点进行系统解读。

## 一、建筑法律法规体系完善——推动建筑法修订

### 改革方向

住建部明确提出"对照高质量发展要求，针对行业发展痛点难点问题，加快推动建筑法等法律法规修订，补齐部门规章短板"。

### 修订重点

1. **理顺管理体制**：完善建筑业管理的体制、机制和法治；
2. **压实主体责任**：理顺建筑市场各方主体的责任、权利和义务；
3. **构建统一大市场**：加快构建全国统一大市场，破除地方保护；
4. **激发市场活力**：更好激发各类市场主体的内生动力和创新活力。

### 政策意义

建筑法自1998年施行以来，虽经2011年和2019年两次修正，但改革力度有限。新一轮修订将以高质量发展为目标，系统性解决行业发展的体制机制障碍。

## 二、建筑业企业资质管理改革

### （一）改革核心方向

1. **简化资质类别**：减少企业资质类别和等级设置；
2. **告知承诺制**：资质审批推行告知承诺，降低企业制度性交易成本；
3. **智能化审批**：推进资质智能化审批，减少人为干扰因素，精简申报材料，压缩审批时限；
4. **弱化人员硬约束**：中级以上职称人员、技术工人不再列为否定资质审核的要件，仅作为衡量企业综合实力的参考指标。

### （二）动态监管强化——"宽进严管"转型

为配合审批简化和准入放宽，住建部着力构建"严管"机制：

1. **资质动态核查**：加大核查频次，对不符合资质标准的企业在全国建筑市场监管公共服务平台标注"资质异常"；
2. **重点监管清单**：对标注企业以及发生安全事故、存在拖欠账款或农民工工资、转包、违法分包等违法行为的企业实施重点监管；
3. **市场清出机制**：依法严肃查处发生质量安全事故、弄虚作假骗取资质的企业或个人。

### （三）注册执业人员管理新规

1. **变更频次公示**：在全国平台公示注册人员变更注册单位次数和轨迹；
2. **限制高频变更**：1年内变更注册单位2次及以上的人员，再次申请注册时不适用告知承诺制；
3. **预警机制**：注册人员注销/变更后，对人员不再达标的企业进行预警提醒；
4. **一人一企原则**：注册人员在2年内只能用于1家企业申请住建部审批的资质。

### （四）各省实践——以陕西省为例

陕西省住建厅2025年6月印发《进一步促进建筑业"稳就业、稳企业、稳市场、稳增长"若干措施》，明确：
- 资质延续5年有效期时，业绩实行**告知承诺制**；
- 由企业法定代表人书面承诺业绩项目技术指标等信息真实合法有效；
- 结合资质动态核查对企业承诺真实性进行核实；
- 失信企业将依法予以严肃查处。

## 三、招标投标市场整治

### （一）整治背景

部分地方存在围标串标、弄虚作假、低价抢标等问题，严重破坏市场公平竞争。住建部明确"开展房屋市政工程招标投标突出问题系统整治"。

### （二）工程服务按质择优导向

1. **提高技术标权重**：优化勘察、设计、监理等工程服务评标方法，突出技术因素，重点评审项目负责人及其团队能力、技术方案；
2. **降低报价权重**：降低投标报价权重，防止恶意低价竞争；
3. **两阶段评审**：技术标和投标报价分别评审——先进行技术标评审，再对通过技术标评审的投标报价进行评审，最后综合评定。

### （三）评标规则改革探索

1. 技术标优先模式
2. 合理低价中标制度
3. 异常低价评审制度

## 四、建筑市场信用体系建设

### （一）全国建筑市场监管公共服务平台功能升级

1. **信息公示**：企业资质、注册人员、工程业绩、信用评价等信息全面公开；
2. **变更轨迹追踪**：注册人员变更轨迹可追溯；
3. **资质异常标注**：不达标企业实时标注"资质异常"。

### （二）失信惩戒机制

1. 发生安全事故、弄虚作假、转包违法分包等行为纳入信用记录；
2. 失信企业在资质审批、招标投标、工程担保等环节受到限制；
3. 对质量安全事故负有责任的项目负责人实施"一票否决"——罚款、停止执业、吊销资格。

## 五、建筑产业工人队伍建设

1. **培养高技能人才**：培养高技能建筑产业工人和高素质专业技术人才队伍；
2. **保障工人权益**：保障农民工工资支付，维护建筑工人合法权益；
3. **化解企业账款**：积极推动化解拖欠建设工程企业账款。

## 六、建筑业转型升级方向

### （一）智能建造与绿色建筑

住建部指导产业基础好的地区：
1. 推进智能建造、绿色建筑等新兴产业培育；
2. 选定骨干企业，建立产学研融合创新机制；
3. 强化科技赋能，打造现代化建筑产业集群。

### （二）现代化建筑产业链

1. 加快培育现代化建筑产业链；
2. 引领产业链上下游协同发展；
3. 提升"中国建造"核心竞争力，发展新质生产力。

### （三）装配式建筑推广

推动建筑工业化、智能化转型，发展装配式建筑，提高建筑产业效率和质量水平。

## 七、政策趋势与展望

### （一）短期趋势（2025-2026年）

1. 资质审批持续简化，告知承诺制进一步推广；
2. 招标投标整治深入开展，多地通报典型案例；
3. 建筑市场信用体系加速建设；
4. 动态监管力度持续加大。

### （二）中期趋势（2026-2028年）

1. 建筑法修订有望完成，为建筑业改革提供法律保障；
2. 全国统一建筑大市场基本形成；
3. 资质管理制度实现从"审批制"向"承诺制+动态监管制"转型；
4. 智能建造和绿色建筑产业规模显著扩大。

### （三）对建筑企业的影响

1. **合规要求提高**：资质标准虽简化，但动态监管更严，企业须持续保持合规；
2. **诚信是基础**：信用体系日趋完善，失信行为将面临多重限制；
3. **技术与人才是核心竞争力**：企业须加强技术创新和人才培养，提升核心竞争力；
4. **竞争更加充分**：市场准入放宽将引入更多竞争者，企业须通过质量、效率、服务赢得市场。

## 核心政策要点总结

1. **法治先行**：建筑法修订是制度基础，理顺体制、压实责任、构建统一大市场；
2. **简政放权**：资质类别简化、审批告知承诺、智能化审批；
3. **放管结合**：宽进不等于弱管，动态核查常态化、异常标注公开化、清出机制刚性化；
4. **质量导向**：招标投标从"拼价格"转向"拼质量"和"拼技术"，按质择优；
5. **诚信为本**：信用档案全覆盖，失信联合惩戒，诚信是企业发展的基石；
6. **转型引领**：智能建造、绿色建筑、装配式建筑引领行业转型升级。
