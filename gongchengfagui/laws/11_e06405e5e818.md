# 《必须招标的基础设施和公用事业项目范围规定》政策解读（大幅缩小强制招标范围）

## 一、出台背景

2018年6月6日，国家发展改革委印发《必须招标的基础设施和公用事业项目范围规定》（发改法规〔2018〕1531号，以下简称"843号文"），作为《必须招标的工程项目规定》（国家发展改革委第16号令，以下简称"16号令"）的配套文件，与16号令同日施行。

此前，原国家发展计划委颁布的《工程建设项目招标范围和规模标准规定》（第3号令，以下简称"3号令"）在施行中逐步显现出范围过宽、标准过低的问题。同时，各省区市地方政府普遍制定了本地区必须招标项目的具体范围和规模标准，进一步扩大了强制招标范围，降低了招标工作效率、增加了企业成本、加重了市场主体负担。针对上述问题，国家发展改革委会同国务院有关部门进行了修订。

## 二、核心改革：大幅缩小强制招标范围

843号文坚持"该放的要放到位，该管的要管住管好"以及"确有必要、严格限定"的原则，将原3号令规定的12大类必须招标的基础设施和公用事业项目，压缩到能源、交通、通信、水利、城建等5大类，大幅放宽对市场主体特别是民营企业选择发包方式的限制。具体体现在以下三个方面：

**（一）删除民间资本投资较多的项目类别**

删除了原3号令中民间资本投资较多的商品住宅项目、科教文卫体和旅游项目、市政工程项目（城市轨道交通除外）、生态环境保护项目等。这意味着在这些领域，市场主体可以根据自身需要自主选择发包方式，不必强制进行招标。

**（二）删除兜底条款**

删除了原3号令中的"其他基础设施项目"和"其他公用事业项目"的兜底条款，避免强制招标范围在执行中被任意扩大。

**（三）保留类别也大幅缩减**

对保留的5大类（能源、交通、通信、水利、城建），与原3号令相比也作了较大缩减。例如，水利类仅保留防洪、灌溉、排涝、引（供）水等基础设施项目；城建类仅保留城市轨道交通项目。

## 三、843号文规定的具体范围

根据843号文第二条规定，不属于16号令第二条、第三条规定情形的大型基础设施、公用事业等关系社会公共利益、公众安全的项目，必须招标的具体范围包括：

1. **能源基础设施项目**：煤炭、石油、天然气、电力、新能源等能源基础设施项目；
2. **交通运输基础设施项目**：铁路、公路、管道、水运，以及公共航空和A1级通用机场等交通运输基础设施项目；
3. **通信基础设施项目**：电信枢纽、通信信息网络等通信基础设施项目；
4. **水利基础设施项目**：防洪、灌溉、排涝、引（供）水等水利基础设施项目；
5. **城建项目**：城市轨道交通等城建项目。

## 四、与原3号令对比的主要变化

| 比较维度 | 原3号令 | 843号文 |
|---------|---------|---------|
| 项目大类 | 12大类 | 5大类 |
| 商品住宅项目 | 必须招标 | 不再强制招标 |
| 科教文卫体项目 | 必须招标 | 不再强制招标 |
| 市政工程（除轨交） | 必须招标 | 不再强制招标 |
| 生态环境保护 | 必须招标 | 不再强制招标 |
| 兜底条款 | 存在 | 删除 |

## 五、制度意义

843号文的出台是招标投标领域深化"放管服"改革的重要举措，有助于扩大市场主体特别是民间投资者的自主权，减轻企业负担，激发市场活力和创造力。同时，与16号令将必须招标项目的规模标准（施工400万元、货物200万元、服务100万元）共同构成完整的必须招标制度体系，实现了招标投标制度的重大变革。

2020年10月，国家发展改革委办公厅又发布《关于进一步做好〈必须招标的工程项目规定〉和〈必须招标的基础设施和公用事业项目范围规定〉实施工作的通知》（发改办法规〔2020〕770号），就准确理解依法必须招标的工程建设项目范围、规范规模标准以下工程建设项目采购、严格执行依法必须招标制度等作出进一步细化规定。
