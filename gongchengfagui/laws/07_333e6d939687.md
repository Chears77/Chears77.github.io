# 广东省工程建设项目招标投标电子交易档案管理办法

> 发布机构：广东省发展改革委牵头，九部门联合 ｜ 文号：粤发改规〔2023〕8号 ｜ 发布日期：2023-09-18 ｜ 实施日期：2023-10-16（有效期5年，现行有效）

> 说明：本办法规范工程建设项目招标投标电子交易档案的归集、存储、保管与共享。以下为按真实要点整理的学习性初稿，正式引用请以广东省政府公报原文为准。

## 第一章 总则

### 第一条

为了规范工程建设项目招标投标电子交易档案管理，保障档案完整、准确、安全、可用，根据有关法律、法规和规定，结合本省实际，制定本办法。

### 第二条

本办法适用于本省依法必须招标的工程建设项目招标投标电子交易档案的管理。

## 第二章 归档要求

### 第三条

招标投标电子交易档案实行"一项一档"，由招标人或者招标代理机构负责归集，并在归档后 30 日内发起归档。

### 第四条

合同电子文件应当在中标通知书发出后 30 个工作日内补充归档。

### 第五条

电子交易档案应当与"广东省招标投标在线监管平台"互联共享，实现数据互通。

## 第三章 存储与保管

### 第六条

归档电子文件应当采用符合规范的数据格式。基本数据类型与格式要求如下：

| 基本数据类型 | 格式 |
| --- | --- |
| 数据库 | DBF |
| 文本 | DOC、DOCX、WPS、TXT、PDF、RTF、OFD |
| 图形 | DXF |
| 图像 | PNG、GIF、TIFF、JPEG、BMP、XBM |
| 图纸 | DWG |
| 清单 | GBQ、COS |
| 音频 | MP3、WAV、MIDI |
| 视频 | RM、AVI、MPEG-2、MPEG-4 |
| 网页 | SGML、HTML、XML |
| 幻灯片 | PPT、PPTX、DPS |
| 表格 | XLS、XLSX、ET |
| 扫描件 | TIFF、JPEG |

### 第七条

音视频档案的保管期限不少于 2 年，其他电子档案的保管期限不少于 15 年。

## 第四章 监督管理

### 第八条

有关行政监督部门应当对电子交易档案的归集、保管、利用实施监督，依法查处违法违规行为。

### 第九条

档案保管单位应当建立健全安全管理制度，确保电子档案不被篡改、不遗漏、可追溯。

## 第五章 附则

### 第十条

本办法自2023年10月16日起施行，有效期5年。
