更新网站方法：
1. 在 `Chears77.github.io` 文件夹内部空白处，右键打开 `Git Bash Here`
2. 一起输入或者分部输入
git add .
git commit -m "更新页面内容，补充文章配置"
git push


更新文章列表方法：
cmd中输入
cd /d E:\Codex\Chears77.github.io\子页文章
node generate-index.js
（或者在文件资源管理器中打开 E:\Codex\Chears77.github.io\子页文章 文件夹，在地址栏输入 cmd 回车，然后再输入 node generate-index.js 回车就行了）
