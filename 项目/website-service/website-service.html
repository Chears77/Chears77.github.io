网页代建服务

帮个人和小企业搭建网站 · 500 元起 · 市场调研中

早期 前端 GitHub Pages 服务类

项目概述

为个人和小型企业提供网页搭建服务，使用 GitHub Pages 作为免费托管平台，帮客户从零到一建立一个可访问的个人主页或企业展示网站。定价从 500 元起，提供从域名配置到页面设计到上线的全流程服务。

风险提示：在副业优先级分析中，AI 已对此项目发出警告——AI 建站工具（V0、Bolt、Lovable）正在快速蚕食这个市场，赛道存在消失风险。但仍值得记录整个思考过程。

想法起源

看到了身边朋友和同学的需求——很多小生意老板、自由职业者、学生社团都想有一个自己的网站，但不知道怎么做。GitHub Pages 是免费的，只需要帮他们写几行 HTML/CSS、配个域名，成本几乎为零，但能收几百块的技术服务费。

而且 GitHub Pages 支持静态网站托管，对于个人博客、企业名片页、活动网页这类需求完全够用。

AI 协作过程

第一轮：市场调研

我：帮人做网站这个方向怎么样？用 GitHub Pages，成本很低
AI：这个方向门槛确实低，但有两个致命问题。第一，AI 建站工具（V0、Bolt）正在快速进步，用户一句话就能生成完整网页，你的 500 元定价会和 AI 的免费方案直接竞争。第二，护城河极弱——你会的东西别人也能用 AI 学会。但如果定位为「熟人圈子的小范围服务」而非规模化的生意，仍然有价值。

第二轮：可行性分析

AI 在整体可行性报告中对此方向做了详细评估，结论是：短期可以做几单赚零花钱，但长期不能作为主攻方向。建议把精力更多地放在护城河更深的项目上（如回忆录代写），网页代建作为补充收入来源。

第三轮：知乎引流尝试

AI 帮忙撰写了知乎文章《帮人做个人网站一个 500 块起步——我在成都想找一个会前端或者会写内容的搭子》，尝试用内容引流获客，同时寻找志同道合的合作伙伴。

产出成果

  • 可行性分析：明确了这个方向的局限性和风险
  • 定价策略：500 元基础版 → 2000 元定制版，区别在于页面数量和复杂度
  • 知乎引流文章：已发布，探索市场反馈
  • 技术方案：GitHub Pages + HTML5 UP 模板 + 自定义域名

最新进度

在整体副业优先级规划中，此项目已被降级。AI 的评估是：AI 建站工具正在消灭这个赛道，投入产出比不高。目前暂不主动推进，保留为熟人圈子的备用选项，如有朋友主动找上门再做。
